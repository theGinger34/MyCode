To execute and check the network connection between client and the server, do the following:
	1)Run the server in one of the computers and select the socket type and the port number.
	2)Run the clients in different computers and the same socket type and port number as the server to establish a connection.
	3)Send messages from the client computers to the server.
	4)The messages get posted on the server.
	5)To end a client's connection to the server, type end.
